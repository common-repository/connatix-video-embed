== Description ==
Easily access, deliver, and monetize your video content directly from WordPress using the Connatix plugin. Built for editors to streamline workflows, natively embed videos within articles without having to log in to the Connatix Platform. 
[Not already a Connatix partner? Get started!](https://connatix.com/contact?utm_medium=landing%20page&utm_source=wordpress&utm_campaign=2022-01-05-wordpress-plugin-page-description)

Key Features: 
* Upload videos to your Connatix Library
* Natively embed videos and playlists from the Connatix CMS
* Leverage dynamic videos from the Connatix Marketplace
* Pull & insert player embed codes onto WordPress pages
* Available for both Classic and Gutenberg WordPress Editors
